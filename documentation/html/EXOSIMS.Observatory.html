WFIRST Observatory class

This class contains all variables and methods specific to the WFIRST observatory needed to perform Observatory Definition Module calculations in exoplanet mission simulation

keepout(time, catalog)[source]¶

Finds Boolean values for each target at current mission time and returns True if successful.

Inputs: current time (MJD), star catalog information Output: True if successful, False if not Finds: Boolean values for each target where True is observable and

False is target unobservable (bright object inside keepout zone)
orbit(time)[source]¶

Finds WFIRST observatory orbit position vector (km) in heliocentric equatorial frame and returns True if successful

Input: current time (astropy Time object) Output: True if successful, False if not Finds: observatory position vector (numpy array) in self.r_sc
