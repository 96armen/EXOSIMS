Lindler Zodiacal Light class

This class contains all variables and methods necessary to perform Zodiacal Light Module calculations in exoplanet mission simulation using the model from Lindler

fbeta(beta)[source]¶

Empirically derived variation of zodiacal light with viewing angle (from Lindler)

Input: beta - angle in degrees Output: f - zodiacal light in zodi

fzodi(simuni)[source]¶

Returns exozodi levels for systems with planets (called in Simulated Universe)

Input: simuni - Simulated Universe object Output: fzodi - exozodiacal light assigned to each planet indexed with planet index
